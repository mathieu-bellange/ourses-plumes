# HTML Tags Usage <>
--------------------------------------------------------------------------------
  <cite> : should be used for the title of a work -- will be used for authors too
    use class .author instead ...
  <summary> : should be used as a caption for a <details> parent element -- will be used for summarize articles
    use class .summary instead ...

# HTML Element ID Syntax and Usage
--------------------------------------------------------------------------------
# Bookmarking
  Syntax      : element-id
  - Target    : <h1 id="element-id">
  - Referer   : <a href="element-id">
  Usage       : inner links (anchors)

# Scripting
  Syntax      : _element-id
  - Target    : <div id="_element-id">
  - Referer   : <a onClick="document.getElementById(_element-id).className = 'active';">
  Usage       : script references (jQuery)
